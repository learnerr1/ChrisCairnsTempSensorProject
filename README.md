# Temperature Sensor Simulation Project

## Project Description
This project simulates temperature sensors

## Features

- Sensor initialised from configuration file
- Temperature data simulation
- Data validation and anomaly detection
- Data logging and storage(file)
- Data smoothing
- Fault simulation

## Installation
1. Clone the repository
2. Ensure .NET SDK is installed
3. in terminal run `dotnet restore`

## Usage
dotnet clean
dotnet build 

dotnet run
